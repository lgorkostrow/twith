# Twith Project
Twith is a simple Twitter clone, build with `ASP.NET Core 5` and `PostgreSQL` 

# Design Patterns
* Domain Driven Design
* Domain Events  
* Clean Architecture
* CQRS
* Repository

# How to run
* `sh build.sh`

# Testing
* Terminal: `dotnet test`

# Swagger (Dev env only)
* http://localhost:5001/swagger
